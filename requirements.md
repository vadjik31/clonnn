# PROCTO 13 LLC - Brand Management System

## Исходное ТЗ
Внутренняя веб-система управления брендами для PROCTO 13 LLC:
- Админ загружает Excel с товарами/ASIN/картинками и брендами
- Система формирует уникальную базу брендов (без дублей)
- Сёрчеры получают бренды пакетами по 100 без пересечений
- Сёрчеры ведут статусы/воронку/заметки
- Админ видит все таблицы, статусы, историю действий, KPI и контроль активности

## Реализованный функционал (MVP)

### Backend (FastAPI + MongoDB)
- ✅ JWT авторизация (email + password + secret_code)
- ✅ CRUD пользователей (админ/сёрчер)
- ✅ Импорт Excel с дедупликацией брендов
- ✅ Атомарная выдача брендов (по 100 штук)
- ✅ Статусы: IN_POOL, ASSIGNED, IN_WORK, WAITING, ON_HOLD, OUTCOME_*, PROBLEMATIC
- ✅ Воронка: REVIEW → EMAIL_1 → EMAIL_2 → MULTI_CHANNEL → CALL_OR_PUSH → CLOSED
- ✅ Заметки с обязательными полями
- ✅ История событий (логирование)
- ✅ Heartbeat для контроля активности
- ✅ Настройки таймингов воронки
- ✅ Админские действия: освободить/переназначить бренд
